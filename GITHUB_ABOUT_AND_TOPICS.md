## Suggested GitHub Pages Setup

1. Go to repository Settings.
2. Open Pages.
3. Select branch: `main`.
4. Select folder: `/root`.
5. Save.
6. Open the generated URL and verify the app loads.

Only use fake demo data for public screenshots and demos.
